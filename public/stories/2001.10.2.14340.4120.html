Okay, okay, so I already have Mandrake 8.0 running right now, and the real reason I'm doing this is because the better of the two web browsers (Konqueror > Mozilla) has decided that it doesn't like running anymore, and I'm hoping a re-install will fix this. Also, I'm beginning to suspect that my e-mail program s getting buggy, but perhaps that's all in my head.

I was going to bring my 3com HomeConnect Webcam here to work with me, but I discovered that it simply won't work. (Well, correction, it would, but I'd have to re-compile my kernel with some alpha-level software and use some freaky command-line software on it.)

Still, this update should be better than what I'm running now, and if I'm lucky, I'll actually be able to get my FireWire hard drive working (so I can really start to use it as the take-anywhere, plug-into-anything solution it was meant to be.)

I spent several minutes downloading the three ISOs with my hundred-meg ethernet connection, and I'm working on getting them burned to a CD (of course, all the working CD-burners in the lab are on either Macs or PCs...)

So, I will continue to work on this, and hopefully, later today, I can be working on a less-unstable, mature beta-level operating system...
